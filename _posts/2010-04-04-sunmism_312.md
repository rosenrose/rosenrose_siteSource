성련선을 살리기 위한 나즈린의 눈물겨운 노력이 돋보이는 좋은 작품이군요(...)<br/>
<br/>
운잔의 눈물겨운 모습이 참...<br/>
<br/>
몇가지 설명 들어갑니다<br/>
<br/>
<span style="font-weight: bold;">제3 함교.</span><br/>
<br/>
허구한날 터져나가는 안습한 장소지요<br/>
<br/>
원 소재는 일본의 애니메이션 우주전함 야마토의 "제3 함교 대파!"의 컷 돌려쓰기입니다.<br/>
<br/>
비슷한 장소로는 언제나 탄막이 옅은 "좌현"(...)<br/>
<br/>
<span style="text-decoration: line-through;">이것은 사실 무라사 선장의 이치린 계획살인</span><br/>
<br/>
실제로 제3 함교의 역할은 2차 세계대전까지의 전투함은 보통<br/>
<br/>
1함교는 전투지휘 함교, 2함교는 항해용 함교, 3함교는 1함교의 예비로 1함교를 축소한 형태로 설치됐답니다<br/>
<br/>
보통은 순양함급 이상에만 3함교까지 설치되었고<br/>
<br/>
평소 항행시에는 2함교에서 함장 이하 선원들이 조타를 하다 
전투시에는 1함교로,<br/>
<br/>
교전중에 1함교가 기능이 마비되면 3함교로 이동하여 작전을 수행했다고 합니다.<br/>
<br/>
<span style="font-weight: bold;"><br/>
도쿄도 타이토구 코마가타 반다이의 완구 제 3 사업부의 별</span><br/>
<br/>
밑에 주석을 달아놓긴 했지만 그래도 설명.<br/>
<br/>
패미컴 소프트인 "세인트 세이야 황금전설"의 치트 패스워드입니다.<br/>
<br/>
원문은 <span style="font-weight: bold;">とうきょうとたいとうくこまがたばんだいのがんぐだいさんぶのほし</span><br/>
<br/>
한자로 표기할 시 <span style="font-weight: bold;">東京都台東区駒形　バンダイ玩具第３部の星</span><br/>
<br/>
해당 패스워드를 입력시 사실상의 최고레벨을 뛰어넘는<br/>
<br/>
40레벨로 게임을 플레이 할 수 있다고 합니다.<br/>
<br/>
그럼 저는 다음 번역으로 찾아뵙겠습니다 :)<br/>
