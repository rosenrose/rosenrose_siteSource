<br/>
이 서클에 등장하는 뱌쿠렌은 거의 사나에급의 상식을 보여주고 있습니다.<br/>
<br/>
예대제7 신간인 나무사로크도 기대되는군요. 과연 얼마만큼의 센스일지(....)<br/>
<br/>
<br/>
제목에 대하여.<br/>
<br/>
<font color="#3a32c3" style="color: rgb(0, 0, 0); font-weight: bold;">そして仏滅 宇留吐羅そぉい!</font> <br/>
<br/>
...한자를 읽을때 음독 훈독 두가지가 있는것을 이용한 일종의 말장난식 제목입니다.<br/>
<br/>
<font color="#3a32c3" style="color: rgb(0, 0, 0); font-weight: bold;">宇留吐羅</font> 그냥 소리나는대로 읽으면 우류토라, 별 의미가 없고 뜻도 요상하지만<br/>
<br/>
저걸 일본식 음독으로 읽는다면 <span style="font-weight: bold;">"우루토라"</span> 즉 <span style="font-weight: bold;">울트라(Ultra)</span> 가 됩니다(...)<br/>
<br/>
(이런표기는 고대시절 폭주족들이 많이 썼다고  합니다. 비슷한 예로 <span style="font-weight: bold;">夜露死苦</span>라고 쓰고 <span style="font-weight: bold;">요로시꾸</span>라고 읽는것도 있지요)<br/>
<br/>
현재 발매되는 만화책 가운데서 찾는다면... .기억나는건<br/>
<br/>
박성우 작가님의 만화인 흑신에 등장하는 히로인<br/>
<br/>
"쿠로"의 익시드인<span style="font-weight: bold;"> 滅牙得救世(메가에그제)</span>가 좋은 예가 되겠군요.<br/>
<br/>
...전작인 츄츄 러블리(생략) 에서도 이런 장난질을 치더니ㅇ&lt;-&lt;<br/>
<br/>
안그래도 허접한 실력에 저런것까지 끼워넣으면 저는 죽어날뿐(...)<br/>
<br/>
그럼 다음 작품으로 찾아뵙겠습니다.<br/>
