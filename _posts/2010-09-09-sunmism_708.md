<br/>
<span style="font-weight: bold;">오오 경사로세 경사로세 자축(...)</span><br/>
<br/>
오늘 구솔은 한결같은 <span style="font-weight: bold;">치르노 사랑</span>으로 <span style="text-decoration: line-through; font-weight: bold;">미친</span> 엄청난 일을 저질렀습니다(...)<br/>
<br/>
올린 시간을 보니 ⑨월⑨일 23시 5⑨분이군요(...) 1분타이밍!<br/>
<br/>
시간에 쫓겨 하다보니 평소라면 다 손댔을 전, 후기는 거의 손도 못댔습니다...<br/>
<br/>
그나마 최저한도의 퀄리티는 유지된 것 같아서 다행이네요.<br/>
<br/>
다음부터는 미리 대비를 하던가 해야지, 아무래도 하루사이에<br/>
<br/>
번역부터 시작해서 식자까지 몰아치는건 미친 짓이었던 것 같습니다<br/>
<br/>
그래도 개인적 한계치를 본 것 같으니 다행(?)...<br/>
<br/>
아무튼 치르노는 최강입니다!<br/>
<br/>
치르노는 바보가 아니에요!<br/>
<br/>
그럼 저는 다음 번역으로 찾아뵙겠습니다 (_ _)<br/>
